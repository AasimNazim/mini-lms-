# Mini LMS - Learning Management System

A web-based Learning Management System (LMS) developed using React that helps teachers and students manage academic activities digitally.

The platform provides an organized environment for managing courses, attendance, assignments, and student performance.

## Features

### Teacher Module
- Create and manage courses
- Mark student attendance
- Upload and manage assignments
- Track student performance

### Student Module
- View enrolled courses
- Submit assignments
- Check attendance records
- View grades and progress

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3
- Firebase (Authentication/Database)
- Responsive Web Design

## Project Workflow

User Authentication → Course Management → Attendance Tracking → Assignment Submission → Performance Monitoring
